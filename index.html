SCOUT

Sistema de reconocimiento de voz.

Scout permite al usuario poder navegar en esta pagina, haciendo uso unicamente de su voz.

TEXTO

En esta sección, todo lo que el sistema - escuche - del usuario será transcrito en la caja de texto.
Para iniciar la transcripción, el usuario debe decir la palabra *ESCRIBIR* (en señal de inicio).

MULTIMEDIA

En esta sección el sistema debe escuchar al usuario, por tal motivo el usuario deve de mandar a realizar una accion.
La acción que obedecerá el sistema corresponde a las palabras "REPRODUCIR" y/o "DETENER" .

GALERIA

En este albúm de imágenes el usuario podra visualizar cada fotografia, indicandole al sistema que dirección debe seguir hacia "ADELANTE" o hacia "ATRAS".
